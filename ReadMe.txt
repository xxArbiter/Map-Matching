========================================================================
    控制台应用程序：Map-Matching 项目概述
========================================================================

应用程序向导已为您创建了此 Map-Matching 应用程序。

本文件概要介绍组成 Map-Matching 应用程序的每个文件的内容。


Map-Matching.vcxproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件，其中包含生成该文件的 Visual C++ 的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

Map-Matching.vcxproj.filters
    这是使用“应用程序向导”生成的 VC++ 项目筛选器文件。它包含有关项目文件与筛选器之间的关联信息。在 IDE 中，通过这种关联，在特定节点下以分组形式显示具有相似扩展名的文件。例如，“.cpp”文件与“源文件”筛选器关联。

main.cpp
    这是主应用程序源文件。

Map-Matching.cpp
    这是 naive map matching 算法源文件

Map-Matching.h
    这是 naive map matching 算法头文件

ST-Matching.cpp
    这是 st-matching 算法源文件

ST-Matching.h
    这是 st-matching 算法头文件

/////////////////////////////////////////////////////////////////////////////
其他标准文件:

StdAfx.h, StdAfx.cpp
    这些文件用于生成名为 Map-Matching.pch 的预编译头 (PCH) 文件和名为 StdAfx.obj 的预编译类型文件。
